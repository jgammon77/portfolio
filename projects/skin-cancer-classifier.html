Skin Cancer Diagnosis Classifier with Decision Trees

A supervised learning project using a Decision Tree classifier to distinguish between benign and malignant skin cancer cases based on clinical features.

Overview

This project uses a dermatology dataset to build a simple yet interpretable model for predicting whether a skin lesion is likely benign or malignant. The focus is on understanding the full ML pipeline: preprocessing, training a Decision Tree, evaluating performance, and explaining feature importance.

While not intended for real clinical use, it demonstrates how classification models can support decision-making and how to work responsibly with health-related data.

Data & Methods

Dataset

  • Tabular dataset with clinical features describing skin lesions.
  • Target label indicating benign vs malignant.

Preprocessing

  • Loaded CSV into pandas and inspected missing values and distributions.
  • Encoded categorical variables as needed (e.g. label encoding / one-hot).
  • Split data into train and test sets (e.g. 80/20).

Modeling

  • Trained a DecisionTreeClassifier using scikit-learn.
  • Tuned basic hyperparameters such as tree depth and splitting criteria.
  • Evaluated using accuracy, precision, recall, and confusion matrix.

Tech Stack

Python, scikit-learn, pandas, NumPy, matplotlib, Jupyter
